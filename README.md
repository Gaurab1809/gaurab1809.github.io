Gaurab1809.github.io is a modern personal portfolio website built to showcase my projects, skills, achievements, and contact information. It is fully responsive, visually appealing, and easy to navigate.

Key Highlights:

Displays personal projects with descriptions and tech stack  
Highlights skills and expertise with interactive UI  
Includes contact section for professional connections  
Fully responsive design for desktop and mobile    
Built with React and modern front-end best practices  

Keywords: personal portfolio, React website, web development, frontend portfolio, responsive design, projects showcase

Steps to Run Locally:

1. Clone the repository:
```bash
   git clone https://github.com/Gaurab1809/gaurab1809.github.io.git
```

3. Navigate into project folder:
```bash
   cd gaurab1809.github.io
```

5. Install dependencies:
```bash
   npm install
```

7. Start local development server:
```bash
   npm start
```

9. Open http://localhost:3000 in your browser to view the website
